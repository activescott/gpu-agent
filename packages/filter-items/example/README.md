# Filter Items Example

A simple React app demonstrating the `@activescott/filter-items` package.

## Quick Start

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Then open http://localhost:5173 in your browser.

## What This Demonstrates

- **FilterItems** component with numeric and categorical filters
- **FilterLayout** for responsive sidebar + content layout
- **applyFilters** utility for client-side filtering
- Bootstrap styling integration
