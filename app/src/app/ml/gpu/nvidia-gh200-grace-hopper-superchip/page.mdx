# NVIDIA GH200-Grace Hopper Superchip Machine Learning Accelerator

The NVIDIA GH200-Grace Hopper Superchip, featuring Hopper architecture, is a cutting-edge AI accelerator. It's particularly notable for being the world's first chip to ship with HBM3e memory, providing significant improvements in memory bandwidth and capacity. This superchip is tailored for the AI market, especially for large-scale models that are memory capacity bound. Its high TDP range indicates its capability for intensive computing tasks.
